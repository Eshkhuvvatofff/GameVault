## 📥 O‘rnatish

Loyihani ishga tushirish uchun quyidagi bosqichlarni bajaring:

```sh
# Repository'ni klonlash
$ git clone https://github.com/YOUR_USERNAME/GameVault.git

# Loyihaga o‘tish
$ cd GameVault

# Zaruriy paketlarni o‘rnatish
$ npm install

# Local serverni ishga tushirish
$ npm run dev
```
